Higher or Lower?

Game Aim:

The aim of the game is to correctly guess all 5 Cards numbers, whether it is higher or lower than the previous. or lower than the current card. You win the game if you correctly guess all 5 Cards. If you make an inccorect guess at any point the game stops and you have to start over.

Game Layout:

  1. 5 Cards are displayed in the screen.
  2. 1st Card number is displayed the other 4 Cards are hidden displayed with a '?'.
  3. Card number can range between 0 to 20

How to Play:

  1. Game Starts:
    • First Card will appear. Following Card is hidden.
  2. Submit a guess:
    • Options of "Higher" and "Lower".
    • Guess if the next Card in the sequence will be higher or lower.
  3. Next Card:
    • New Card is revealed after submitting guess.
  4. You will either:
    • Correct Guess:
      • Move onto the next round, Card will turn green.
      • New Card take another guess.
    • Incorrect Guess:
      • Game ends, Card will turn red.
      • If you would like to play again click 'Play Again'. Game starts over guess the next Card must start again from the first card.

Winning the Game:

If you guess all 5 Cards correctly you win the game

Losing the Game:

If you guess incorrectly at any point the game ends you will have to reset the game and try again
